# 🛍️ MOON Clothing – Luxury eCommerce Website

A modern and responsive luxury clothing eCommerce website built using **HTML, CSS, JavaScript, and Bootstrap**.
This project showcases a complete frontend shopping experience with cart functionality, product filtering, and elegant UI design.

---

## ✨ Features

- 🏠 Modern Homepage with Hero Section
- 🛒 Shopping Cart (LocalStorage based)
- 👕 Product Listing & Categories
- 🔍 Filter & Sort Products
- 👀 Quick View Modal
- 📦 Add to Cart Functionality
- 💳 Checkout Page UI
- 📱 Fully Responsive Design
- 🎨 Luxury UI with smooth animations

---

## 🛠️ Tech Stack

- HTML5
- CSS3 (Custom + Bootstrap 5)
- JavaScript (Vanilla JS)
- Font Awesome Icons
- Google Fonts
